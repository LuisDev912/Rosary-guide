# Rosary Guide

A modern, accessible and multilingual guide to praying the Holy Rosary.
Built with React + Vite, fully responsive and designed for clarity and simplicity.

---

## Features

- Step-by-step guide on how to pray the Rosary
-Full list of Mysteries (Joyful, Sorrowful, Glorious, Luminous)
- Multilingual support (ES/EN) using i18next
- Mobile-first responsive design
- Clean and elegant Marian-inspired color palette

---

## Stack

- React
- Vite
- i18next
- CSS mobile-first
